# FT_CONTAINERS

## Mandatory Part
- vector
- map
- stack
- iterators_traits
- reverse_iterator
- enable_if
- is_integral
- equal and / or lexicographical_compare
- std::pair
- std::make_pair

### Requirements
- namespace must be ft
- use std::allocator
- friend keyword must be justified
- map::value_compare (friend allowed)

## Bonus
- set with RB Tree
